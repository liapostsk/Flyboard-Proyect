# Flyboard Agent Router API

## Tech Stack
- Python 3.10+
- FastAPI
- OpenAI Responses API
- SQLite

## Setup
1. Clone repo
2. `pip install -r requirements.txt`
3. `cp .env.example .env` y edita
4. `python -m pytest` para tests
5. `python -m uvicorn app.main:app --reload` para servidor
